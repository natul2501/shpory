Information about the security of the files

-rw-r--r-- keeper prim 547 9:31 chimps
- - it's a file
d it's a directory
rw- rwx "rwx"-group (rwx rwx rwx):
1) "rw-" - user (owner) ("u");
2) "r--" - group ("g");
3) "r--" - other ("o");
r means you can "read" the file's contents
w means you can "write", or modify, the file's contents
x means you can "execute" the file (only if the file is a program).
If any of the "rwx" characters is replaced by a '-', then that permission has been revoked.
keeper UserID
prim the group owner of the file
547 the size of the file in bytes
9:31 the time the file was last modified
3 chmod = change mode; change the security permissions on files
chmod o if you want to give "rwx" permission to the world ("other")
chmod g+x if you want to give "execute" permission to the group
chmod u+w gorillas if you want to give "write" permission to the user ("owner") for file "gorillas"
chmod ugo-rwx gorillas change multiple permissions at once
