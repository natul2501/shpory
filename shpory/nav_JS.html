Шпори по Java Script


Способи виводу інформації - приведені всі відомі мені можливості виводу інформації у вікно браузера, які надає Java Script.

Типи даних - приведені всі відомі мені типи даних, які існують у Java Script, а також перетворення одного типа в інший, оператори JS, арифметика JS, побітові операції, строгий режим JS (ECMAScript 5).

Умови і цикли - приведені всі відомі мені умовні оператори і цикли, які існують у Java Script (if, else, switch, case, default, break, for, in, while, do, continue), а також мітка.

Ключові слова - приведені деякі особливі ключові слова, які існують у Java Script (this, let, const)

Робота із отладчиком - приведений приклад роботи із отладчиком Java Script

Об'єкт String() - приведені всі відомі мені властивості і методи об'єкту String()

Об'єкт Number() - приведені всі відомі мені властивості і методи об'єкту Number()

Об'єкт Math() - приведені всі відомі мені властивості і методи об'єкту Math()

Об'єкт Array() - приведені всі відомі мені властивості і методи об'єкту Array()

Об'єкт Date() - приведені всі відомі мені властивості і методи об'єкту Date()

Об'єкт Error() - приведені всі відомі мені властивості і методи об'єкту Error(), а також перехват помилок, валідація форм (методи DOM), строгий режим(use strict).

Регулярні вирази JS (RegExp) - приведені всі відомі мені регулярні вирази JS: модифікатори регулярних вразів, шаблони, метасимволи, квантифікатори.

Робота з DOM - доступ до елементів на сторінці та їх редагування з використанням об'єкту document.

Робота з BOM - отримання інформації про елементи браузера, способи відкриття веб-сторінки, взаємодія з діалоговими вікнами, пошук на сторінці, друк сторінкию.

Робота з формами в JS - форми та їх елементи (поля вводу, кнопки, перемикачі), їх властивості і події.
