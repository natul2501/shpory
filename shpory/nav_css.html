Шпори по CSS


Стилізація фону - приведені всі відомі мені можливості роботи із стилізацією фону, які надає CSS (колір фону, зображення, градієнт).

Стилізація тексту - приведені всі відомі мені можливості стилізації тексту, які надає CSS (стилізація тексту, маркованих списків, додавання іконок та прихованих блоків).

Стилізація рамок і полів - приведені всі відомі мені можливості стилізації рамок (border, outer) і полів(margin, padding), які надає CSS.

Позиціонування - приведені всі відомі мені можливості позиціонування, які надає CSS, а також властивості обтікання, видимості і властивості зображень, що виходять за межі (position, display, visibility, float, clear, overflow, z-index).

Селектори - приведені всі відомі мені можливості роботи із селекторами, які надає CSS (задання стилів селекторам, комбінування селекторів, застосування псевдокласів і псевдоелементів).

Стилізація таблиць - приведені всі відомі мені можливості стилізації таблиць які надає CSS.

Приклади застосування стилів: Вертикальне меню, Горизонтальне меню, Випливаючі віконця.
