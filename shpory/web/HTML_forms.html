Поля вводу
тег input підтримує атрибути name, reqired, placeholder
<input/> - просто поле input
<input type="text"/> - стандартне текстове поле
<input type="text" size="30"/> - задання ширини поля
<input type="text" required/> - текстове поле с обов'язковим заповненням
<input type="text" placeholder="AAA" /> - текстове поле c плейсхолдером
<input type="text" value="BBB"/> - текстове поле з текстом за замовчуванням
<input type="password"/> - текстовое поле з прихованими символами вводу
<input type="number"/> - цифровой лічильник, можна вводити з клавіатури
<input type="search"/> - текстове поле пошуку
<input type="date"/> - текстове поле з датою
<input type="email"> - текстове поле для введення емейлу
<input type="tel"/> - текстове поле для введення номеру телефону
<input type="datetime"/> - текстове поле для введення дати і часу
<input type="hidden" value="F"/>
поле, яке не відображається у вікні браузера воно використовується для передачі різної допоміжної інформації, наприклад, імені сторінки, на якій знаходиться користувач, або електронна адреса, куди має пересилатися форма

Випадаючі списки
<input list="CCC"> <datalist id="CCC"> <option value="CCC1"></option> <option value="CCC2"></option> <option value="CCC3"></option> </datalist> - поле input з випадаючим списком варіантів, можна вводити з клавіатури
<select name="D1"> <option value="D11">DDD1</option> <option value="D12">DDD2</option> </select> - поле select з випадаючим списком варіантів, можна вводити з клавіатури
<select name="D2" size="2"> <option value="D21">DDD3</option> <option value="D22" selected="">DDD4</option> <option value="D23">DDD5</option> </select> - поле select, в якому задане число видимих рядків; пункт DDD4 вибраний за замовчуванням
<select name="D3" size="2" multiple> <option value="D31">DDD6</option> <option value="D32">DDD7</option> <option value="D33">DDD8</option> </select> - поле select з можливістю вибору декількох варіантів з утриманням клавіші Ctrl
<select name="E"> <optgroup label="E1"> <option value="E11">E11</option> <option value="E12">E12</option> </optgroup> <optgroup label="E2"> <option value="E21">E21</option> <option value="E22">E22</option> </optgroup> </select> - поле select з випадаючим списком варіантів, розбитих по категоріям

Найменування групи LLL1 LLL2 LLL3 - поле перемикачів, згрупованих за допомогою fieldset.
<fieldset form="myForm"> <input type="radio" value="N1">N1 <input type="radio" value="N2">N2 <input type="radio" value="N3">N3 <input type="submit" value="Send"> - пояснення </fieldset>
N1 N2 N3 - поле, що пов'язує групу з формою по id
