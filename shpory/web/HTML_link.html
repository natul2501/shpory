<a href="HTML_link.html" rel="author">Текст посилання</a> – вказує на те, що наведене посилання стосується інформації про автора цієї сторінки або статті

<a href="HTML_link.html" rel="help">Текст посилання</a> – посилання на документ-довідку

<a href="HTML_link.html" rel="next">Текст посилання</a> – поряд із first, up, prev, last використовується для вказування шляху пересування по карті сайту або розділам сторінки
<a href="HTML_link.html" rel="nofollow">Текст посилання</a>
посилання на документ, що не індексується використовується в Google, щоб вказати, що пошуковий бот не повинен переходити за цим посиланням. В результаті сторінка, на яку посилається користувач, не збільшує свою вагу завдяки тому, що не передається індекс цитування, що використовується Яндексом, та PageRank, який використовується Google.
<a href="HTML_link.html" rel="noreferrer">Текст посилання</a> - вказівка браузеру не надсилати HTTP-заголовок, якщо користувач перейшов за посиланням
<a href="HTML_link.html" rel="prefetch">Текст посилання</a>
вказує, що слід кешувати цільовий документ Посилання з такими значеннями стануть у нагоді в тих випадках, коли потрібно послатися на зовнішні ресурси, які користувач відкриє з великою ймовірністю. Браузер кешує це посилання заздалегідь, тому воно відкривається швидше
<a href="HTML_link.html" rel="search">Текст посилання</a> - цей тип повідомляє, що посилання веде до інтерфейсу пошуку
<a href="HTML_link.html" rel="tag">Текст посилання</a> – визначає категорію сайту або ключовий запит
<a href="HTML_link.html" rel="canonical">Текст посилання</a> - вказує на бажану адресу, яка братиме участь у пошуку. Використовується в тезі <header>
<a href="HTML_link.html" rel="external">Текст посилання</a>
означає, що посилання веде на інший сайт Цей сайт індексуватиметься та передаватиме вагу. У WordPress цей атрибут часто використовується у коментарях. Може функціонувати спільно з nofollow (rel="external nofollow"), щоб посилання не передавало вагу
