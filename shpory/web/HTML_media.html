ЗОБРАЖЕННЯ
Перелік основних властивостей
*При наведенні мишкою на зображення відобразиться синтаксис
width та height - задають розмір title - вспливаючий напис alt* - альтернативний текст
draggable="false" - не можна перемістити draggable="true" - можна перемістити style="opacity: 0.6" - задати прозорість css
напівпрозорість при наведенні мишкою css картинка в якості посилання частина картинки (вимикач) в якості карти зображення <map>
В залежності від форми області карти-зображення по-різному задаються її координати. Форми бувають:
circle - коло | default - вся область| poly - багатокутник| rect - прямокутник
ВІДЕО
Перелік основних властивостей
*При наведенні мишкою на відео відобразиться синтаксис
width та height - задають розмір poster - задає посилання на прев'ю muted - беззвучний за замовчуванням
loop - безкінечне відтворення autoplay - автовідтворення відео з субтитрами
відео, додане за допомогою тега <object> відео, додане за допомогою тега <embed> так можна додавати декілька відео під ряд

Перелік основних властивостей
*При наведенні мишкою на описання відобразиться синтаксис

- стандартна аудіопанель

- ще один спосіб виведення аудіо

- аудіо з безкінечним відтворенням

- аудіо з автовідтворенням

- завантаження аудіо після натиснення "play"

- збір інформації про аудіо: розмір, перший кадр, список відтворення, тривалість
