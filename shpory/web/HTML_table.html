Перелік властивостей таблиць:
 - align - вирівнювання таблиці
 - bgcolor - колір фону таблиці
 - border - товщина ліній межі навколо таблиці (також відображаються межі між клітинками)
 - cellpadding - відстань між межею клітинки і її вмістом
 - cellspacing - відстань між зовнішніми межами клітинок
 - width - ширина
 - caption - заголовок
 - colspan - об'єднання клітинок по горизонталі, rowspan - по вертикалі
 - valign - вертикальне вирівнювання
 - Cthrl+Shift+D - скопіювати рядок
