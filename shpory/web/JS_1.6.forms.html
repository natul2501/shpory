Если в HTML-документе определена форма, то она доступна сценарию JavaScript как объект, входящий в объект document с именем, заданным атрибутом NAME тега FORM.

Form

	<form name="my" id="forma">
		<input name="one" value="1">
		<input name="two" value="2">
	</form>
1 document.forms.my; - доступ к форме с именем "my"
2 document.forms[0]; - доступ к первой форме в документе
3 form.elements.one; - доступ к элементу формы с именем "one"
* может быть несколько элементов с одним и тем же именем, это часто бывает с кнопками-переключателями
4 form.one; - сокращённая форма доступа к элементу формы с именем "one"
5 one.form; - обратная ссылка

Свойства формы
1 form.action; - значение атрибута action: имя программы, которая будет обрабатывать форму
2 form.target; - значение атрибута target: фрейм, куда отправить полученную информацию (задается во фреймосодержащих документах)
3 form.method; - значение атрибута method: метод передачи данных от формы обрабатывающей программе; может принимать значения get или post
4 form.encoding; - значение атрибута enctype: тип содержимого (метод кодирования)
5 form.elements; - массив всех элементов формы
6 form.length; - размер массива elements

<fieldset> как «подформа»

	<form name="my2">
		<fieldset name="userFields">
			<input name="three" 	value="3">
			<input name="four"	value="4">
		</fieldset>
	</form>
1 form.elements.userFields.elements.three; - доступ к элементу через fieldset
2 form.elements.three; - доступ к элементу через form

Элементы формы

	<form id="my2">
		<fieldset name="userFields">
			<input name="three" 	value="3">
			<input name="four"	value="4">
		</fieldset>
	</form>
1 form.elements.userFields.elements.three; - доступ к элементу через fieldset
