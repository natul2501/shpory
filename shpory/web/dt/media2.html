Тег <source> используется для указания расположения медийных ресурсов (файлов) для медиа-элементов. Для указания нескольких альтернативных аудио/видео файлов, для обозначения того, что они доступны в нескольких форматах, можно использовать несколько элементов source.

Тег <object> используется для вставки таких объектов как изображения, аудио, видео, java-апплетов, activeX-элементов, PDF и flash, а также чтобы вставить другую веб-страницу в ваш HTML-документ.

Тег <iframe> определяет встроенную область (рамку), которая может отображать содержимое отдельно взятого документа или даже другого веб-сайта в ограниченном пространстве на текущей веб-странице.

Тег <embed> определяет контейнер для отображения объектов (видеофайл, флэш-ролики, звуковые файлы и тд.). Некоторые объекты могут требовать подключения к браузеру специального модуля (плагин).

Тег <canvas> (в переводе с англ. означает холст) создает область на веб-станице, которая может быть использована для отрисовки графики в режиме реального времени с помощью сценариев
