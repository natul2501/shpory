Свойство clear

clear: left

1. Блок должен размещаться ниже всех левосторонних плавающих блоков. Это удобно, если нам не нужно, чтобы текст обтекал блоки справа: его нужно поместить в блок со стилем clear: left

clear: right

2. Блок должен размещаться ниже всех правосторонних плавающих блоков

float: left 3.1
float: left 3.2
float: right 3.1
float: right; clear: both
float: right 3.3

3. Блок должен размещаться ниже всех плавающих блоков

clear: none

4. Никаких ограничений на положение блока относительно перемещаемых объектов не накладывается
