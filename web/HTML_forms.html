Формы

Все упомянутые ниже теги будут корректно отображаться на веб-странице, однако если попытаться обработать соответствующей программой, то ничего не получится. Это произойдет, потому что всем полям формы необходимо присвоить имена. Имена присваиваются при помощи атрибута name, который необходимо добавить каждому полю формы.

Имя:

- просто поле input.

- стандартное текстовое поле.

- текстовое поле с обязательным заполнением.

- текстовое поле c плейсхолдером.

- текстовое поле с текстом по умолчанию.

- текстовое поле со скрытыми вводимыми символами.

- цифровой счётчик, можно вводить с клавиатуры

- текстовое поле поиска

- поле input с выпадающим списком вариантов: можно вводить с клавиатуры.

- поле select с выпадающим списком вариантов: нельзя вводить с клавиатуры.

- поле select, в котором задано число видимых строк; пункт DDD4 выбран по умолчанию

- поле select с возможностью выбора нескольких пунктов меню при удержании клавиши Ctrl

- поле select с выпадающим списком вариантов, разбитых по категориям

- текстовое поле с датой.

- текстовое поле с временем.

- поле, которое не отображается в окне браузера; оно используется для передачи различной вспомогательной информации, например, имени страницы, на которой находится пользователь, или электронный адрес, куда должна пересылаться форма.

- поле, которое позволяет загружать файл с ПК на сервер.

- кнопка.

- кнопка отправки данных формы программе-обработчику.

- кнопка отправки данных формы программе-обработчику с заданной надписью.

- кнопкa, которая при нажатии на нее возвращает все поля формы к значениям по умолчанию.

- кнопка отправки, в качестве которой используется активное изображение.

- кнопка выбора цвета

- бегунок для регулирования величины.

Поле выключателей (checkbox) или флажков: можно либо включить, либо отключить.

Заголовок MMM1 MMM2 MMM3 - группа переключателей с поясняющей надписью

N1 N2 N3 - поле, которое связывает группу с формой по id
