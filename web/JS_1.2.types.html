Побитовые операции

5 = 0101; 1 = 0001
& - побитовое "и" - возвращает 1, если оба бита равны 1
| - побитовое "или" - возвращает 1, если хотя бы один из двух битов равен 1
^ - исключающее "и" - возвращает 1, если только один из двух битов равен 1, но не оба
~ - побитовое "не" - инвертирует все биты
<< - сдвиг влево с заполнением появившихся младших (справа) битов нулями
>> - сдвиг вправо с копированием знака (арифметический сдвиг вправо); пустые биты заполняются копией крайнего левого бита
>>> - сдвиг вправо с заполнением появившихся старших (слева) битов нулями
